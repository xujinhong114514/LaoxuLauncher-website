Laoxu Minecraft Launcher

一个基于 HMCL 开发的 Minecraft 启动器，新增基岩版（Windows 10/11）启动支持。

✅ Java 版完整保留 HMCL 所有功能  |  ✅ 基岩版自动检测  |  ✅ 双版本图形化切换

✨ 主要特性

Java + 基岩版双支持

顶部按钮一键切换，Java 版完整功能，基岩版自动扫描 XboxGames 路径。

成就系统

v1.0.3 新增成就功能，记录你的游戏历程。

基岩版自动检测

自动扫描 C:/D:/E: 等磁盘的 XboxGames 文件夹，无需手动配置。

双版本图形化卡片

全新卡片式界面，Java/基岩版切换更直观。

免安装 · 全平台通用

⚠️ 基岩版仅支持 Windows 10/11，且需要已安装 Minecraft for Windows。
🐧 Linux 用户：chmod +x LaoxuLauncher.sh && ./LaoxuLauncher.sh | 🍎 macOS 用户：java -jar LaoxuLauncher.jar

🚀 使用说明

1下载对应版本（.exe / .jar / .sh / .zip）
2运行启动器，点击顶部按钮切换 Java版 / 基岩版
3点击「启动」按钮开始游戏
💡 基岩版要求：已安装 Minecraft for Windows，启动器会自动扫描所有磁盘分区的 XboxGames 文件夹。
💡 Linux 需要 Java 17+：sudo apt install openjdk-17-jre (Debian/Ubuntu)
